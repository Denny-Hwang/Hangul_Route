# docs/beta/

한글학교 **베타 사이클** 운영 기록.

## 사이클 (반복 단위)
1. **알파** — 내부 빌드, W4 전후.
2. **Pilot 모집** — 1–2개 한글학교, 5–10명 어린이.
3. **필드 세션** — 2주. 주 1회 원격 관찰 + 교사 피드백.
4. **리포트** — `docs/beta/cycle-NN-report.md`.
5. **반영 스프린트** — 리포트 기반 백로그 → `docs/tasks/INBOX.md` 로 투입.

## 파일 네이밍
- `cycle-NN-plan.md` — 사이클 계획 (모집 기준, 목표, 측정 지표)
- `cycle-NN-report.md` — 사이클 리포트 (결과, 배운 점, 다음 액션)

## 측정 지표 (예시)
- Stage 1 Episode 완료율
- 호야 피드백 후 재시도율
- 학부모 인터뷰 한마디
- 세션 당 평균 플레이 시간 (목표 12–15분, 너무 길면 부담)

## 타겟 확인
- 한글학교 학생 = 대다수 P4 (heritage).
- 알파 단계에서 P5 (K-컬처 관심 해외 어린이) 는 개별 모집 (가족 네트워크).
