# docs/roadmap/

**전방 계획** 디렉토리. 회고는 `docs/weekly/` 에 분리되어 있다.

## 구성 (예정)
- `2026-q2.md` — 분기 OKR / 마일스톤
- `milestones.md` — 주요 이정표 (알파, 베타, v1)
- `backlog-themes.md` — 큰 덩어리 테마 (Stage 축, 공간 축, 플랫폼)

## 관계
- `docs/roadmap/` → 장기·전방
- `docs/tasks/INBOX.md` → 단기·집행 가능
- `docs/weekly/` → 과거·회고
- `docs/beta/` → 사이클별 실사용자 피드백

Roadmap 항목은 구체화 단계에서 `docs/specs/F-XXX-*.md` 로 쪼개지고, 그 중 집행 가능한 것만 `INBOX.md` 로 넘어간다.
