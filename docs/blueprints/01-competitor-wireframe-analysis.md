# 한국어 학습 앱 경쟁 와이어프레임 분석 보고서

> **프로젝트명(가칭)**: *KoreanKids* (최종 네이밍 추후)
> **작성 기준일**: 2026년 4월
> **타겟 페르소나**: P4 재외동포 2·3세 어린이 + P5 K-컬처 관심 해외 어린이 (할로우 MVP)
> **제품 비전**: *"한글을 재미있고 쉽게, 꾸준히 성취감 있게 배우는 어린이 서비스"*

---

## 1. 개요 및 방법론

### 1.1 왜 이 3개 앱인가

제품 비전의 세 축 — *재미있게 · 쉽게 · 꾸준히* — 각 축에서 업계 최고 사례를 벤치마킹하기 위해 아래 3개 앱을 선정했다.

| 축 | 선정 앱 | 선정 이유 |
|---|---|---|
| **꾸준히** | Duolingo | 리텐션·스트릭 메커닉의 글로벌 표준. 2022년 선형 Path 개편으로 학습 동선 재정의. 한국어 학습자 1,740만명. |
| **쉽게 (어린이 특화)** | DinoLingo | 어린이 전용(2~14세) 언어 학습 앱 중 가장 구조화된 제품. 50개 언어 중 하나로서의 한국어 현실을 직접 경쟁. 가족 프로필·부모 대시보드 참조. |
| **재미있게 (시각적)** | Drops | 5분 세션 디자인의 교과서. 미니멀 일러스트 + 스와이프 인터랙션. Kahoot! 자회사. |

보완 참고: **한글이야호**(EBS) · **뽀로로 한글놀이** — 한국 Heritage 시장이 실제 소비하는 콘텐츠. 분석 섹션 5에 간략히 다룸.

### 1.2 분석 프레임

각 앱을 5개 레이어로 분해했다:
1. **온보딩**: 첫 열기 → 첫 레슨 시작까지
2. **홈 화면**: 진입 후 학습 루틴의 시작점
3. **레슨 화면**: 실제 학습 단위 내부 인터랙션
4. **완료·보상 화면**: 세션 종료 후 피드백·동기 부여
5. **지속 장치**: 스트릭·알림·소셜·구독 등 리텐션 메커닉

## 2. Duolingo 분석

### 2.1 제품 개요
- **포지셔닝**: 세계 최대 게임화 언어 학습 앱. 성인·청소년·고급 학습자 중심.
- **가격**: 무료 + Super Duolingo 프리미엄 (~$6.99/월)
- **한국어 학습자**: 전 세계 약 1,740만명 (최대 규모)
- **핵심 철학**: "학습은 루틴. 매일 5분 연속이 한 번 1시간보다 낫다."

### 2.2 온보딩 와이어프레임
```
[Screen 1] 언어 선택
 ├─ "What do you want to learn?" (영어 UI 기본)
 ├─ 국기 아이콘 그리드 (스크롤)
 └─ 한국어 선택

[Screen 2] 학습 이유 선택
 ├─ "Why are you learning Korean?"
 ├─ 5개 옵션 (여행/일/학업/가족/재미)
 └─ 각 옵션은 이모지 + 한 줄 텍스트

[Screen 3] 일일 목표 시간 설정
 ├─ Casual 5min / Regular 10min / Serious 15min / Intense 20min
 └─ 기본값: Regular 10min

[Screen 4] 레벨 진단
 ├─ "I'm new to Korean" OR "I know some Korean"
 └─ 후자 선택 시 placement 테스트

[Screen 5] 첫 레슨 즉시 시작
 └─ 가입 요구 없이 바로 학습 (나중에 프로그레스 저장 유도)
```

**핵심 설계 철학**: *"Value first, account later"* — 첫 레슨을 먼저 경험시키고 나중에 가입 유도.

### 2.3 홈 화면: 선형 Path
- **수직 스크롤 길**: 위에서 아래로 레슨 노드(동그란 아이콘)가 이어짐. 지그재그 배치로 시각적 리듬.
- **잠금 구조**: 현재 레슨만 하이라이트, 다음 레슨은 회색, 완료된 레슨은 금색.
- **유닛 구분**: 5~8개 레슨이 한 유닛. 유닛 경계마다 캐릭터(Duo, Zari 등) 일러스트 삽입.
- **상단 HUD**: 연속 학습일(🔥 스트릭), 보석(💎 XP 화폐), 하트(❤️ 실수 허용 횟수), 언어 전환.
- **하단 탭**: Learn · Stories · Leagues · Shop · Profile

**핵심 설계 철학**: *"다음에 할 것을 생각하지 말고 그냥 따라와"* — 선택지를 없애 인지 부담 제거.

### 2.4 레슨 화면
- **상단 진행 바**: 현재 레슨의 몇 %
- **중앙 연습 영역**: 문제 유형이 랜덤 순환
  - 단어-이미지 매칭
  - 빈칸 채우기 (단어 타일 드래그)
  - 번역 (영→한, 한→영)
  - 듣기 받아쓰기
  - 발화 (마이크 입력)
- **하단 CHECK 버튼**: 답안 확정 → 정답/오답 피드백 → Continue
- **오답 시**: 빨간 배너 + 정답 표시 + 하트 -1
- **연속 정답 시**: "Nice work!" 토스트 애니메이션

### 2.5 완료·보상 화면 시퀀스
```
[1] Lesson Complete 축하 화면
    ├─ XP 획득량 (+10 XP, +5 bonus)
    └─ 캐릭터 댄스 애니메이션

[2] 진행도 업데이트
    └─ 이번 주 목표 대비 진행률 바

[3] 리그 포인트 표시 (소셜)
    └─ 리그 내 순위 변화

[4] 스트릭 확장
    └─ "🔥 N일 연속!" 달력 시각화

[5] Continue 버튼으로 홈 복귀
```

### 2.6 지속 장치
- **스트릭 보호 (Streak Freeze)**: 하루 놓쳐도 불꽃 유지 (프리미엄 또는 보석 구매)
- **푸시 알림**: "Duo가 슬퍼해요" 시리즈 (바이럴 밈이 된 그 알림)
- **리그**: 주간 XP 리더보드 (Bronze→Diamond), 실력이 아닌 참여도 랭킹
- **Super Duolingo**: 하트 무제한, 광고 제거, 개인화 복습
- **Duo 캐릭터**: 마스코트로서 감정적 연결 (Duo 사망 마케팅 캠페인 등)

### 2.7 강점 / 약점 (어린이용 관점)

| 강점 | 약점 |
|---|---|
| 리텐션 메커닉 업계 최고 | 한국어 음성이 로봇처럼 부자연스러움 (외부 평가 2/5점) |
| Path 구조의 단순함 | 문법 설명 없음 |
| 게임화 요소 풍부 | 쓸모없는 예문 ("거북이가 양상추를 먹는다") |
| 무료 진입 장벽 낮음 | UI가 성인 취향, 어린이에겐 딱딱 |
| Stories로 맥락 학습 지원 | 한글 자모 학습 부실 |
| | Duo 캐릭터가 때로 위협적(밈) — 5~8세에 부적합 |

### 2.8 Duolingo에서 훔칠 것
1. **선형 Path의 단순함** — 어린이에게 선택지 주지 말 것
2. **스트릭 메커닉** — 단 "죄책감 유발" 방식은 피하고 "성취 축하" 방식으로
3. **세션 완료 후 여러 보상 화면 시퀀스** — 도파민 단계별 발사
4. **연속 정답 칭찬 토스트** — 작은 성취감의 누적

## 3. DinoLingo 분석

### 3.1 제품 개요
- **포지셔닝**: 어린이 전용 다국어 학습 앱 (2~14세)
- **가격**: $14.95~$19/월, $199/년, 7일 무료 체험, 6개 프로필/계정
- **언어**: 50개 (한국어 포함) — 각 언어는 200개 핵심 단어·구문 중심
- **핵심 철학**: *"Total immersion — no translation"* (몰입식, 번역 없음)

### 3.2 온보딩 와이어프레임
```
[Screen 1] 가입/로그인
 └─ 부모 이메일 + 비밀번호 (어린이 독자 사용 전제)

[Screen 2] 자녀 프로필 생성
 ├─ 이름, 나이만 입력
 └─ 아바타(공룡) 선택

[Screen 3] 학습 언어 선택
 └─ 국기 그리드 (50개)

[Screen 4] 어린이 홈으로 진입
 └─ 추가 질문 없이 바로 콘텐츠 노출
```

**핵심 설계 철학**: *"어린이는 질문 답하기 싫어한다"* — 온보딩 질문 최소화.

### 3.3 홈 화면: 테마 그리드
- **상단**: 공룡 마스코트, 오늘의 보상 진행도
- **중앙**: 테마 타일 그리드 (Animals, Food, Colors, Numbers, Family 등)
- **타일 내부**: 큰 일러스트 + 테마명 + 별 획득 수
- **하단 탭**: Home · Videos · Games · Songs · Stories
- **부모 모드 진입**: 숨겨진 버튼 (오른쪽 상단), PIN 입력 필요

### 3.4 레슨 화면 (테마 진입 후)
```
[Step 1] 애니메이션 비디오 시청 (30초~1분)
 └─ 공룡이 해당 주제 단어를 반복 발화

[Step 2] 단어 소개 (플래시카드식)
 └─ 큰 사진 + 원어민 음성 + 자동 진행

[Step 3] 매칭 게임
 └─ 왼쪽 단어 ↔ 오른쪽 이미지 연결 (드래그 or 탭)

[Step 4] 퀴즈
 └─ 3지선다, 오답해도 부드러운 피드백

[Step 5] 보상
 └─ 별 획득 + 새 공룡 스티커 잠금 해제
```

### 3.5 지속 장치
- **부모 대시보드**: 각 자녀의 학습 시간 · 완료 레슨 · 스트릭 · 영역별 정확도 — 실시간 인사이트.
- **가족 스트릭**: 6명 프로필의 주간 합산 스트릭 (가족 단위 게임화)
- **계절 스킨**: Winter Map 등 계절 이벤트
- **오프라인 자료**: 인쇄 가능 워크시트, 플래시카드, 포스터
- **랜덤 공룡 획득**: 다른 언어 테스트 완료 시 보상

### 3.6 강점 / 약점

| 강점 | 약점 |
|---|---|
| 어린이 전용 UI·UX 완성도 | 한국어가 50개 중 하나 — 깊이 부족 |
| 부모 대시보드 완성도 높음 | 한국 문화·맥락 전무 (학교·명절·가족 관계 없음) |
| 가족 프로필 (최대 6명) | 한글 자모 학습 체계 없음 |
| 번역 없는 몰입식 학습 | 200개 단어 수준 이후 성장 불가 (초등 고학년에 부족) |
| 인쇄물로 오프라인 확장 | 구독료 높음 ($19/월) |
| Ad-free, 개인정보 최소 수집 | 레슨이 비디오 의존적, 인터랙션 얕음 |

### 3.7 DinoLingo에서 훔칠 것
1. **부모 대시보드 구조** — 학부모를 Secondary User로 설계하는 것이 어린이 앱의 필수
2. **가족 프로필** — 재외동포 가정은 형제자매 다수 케이스 많음
3. **테마 그리드 홈** — 어린이는 선형 Path보다 "선택할 수 있는 주제 카드"를 더 좋아할 수 있음 (A/B 테스트 가치 있음)
4. **오프라인 인쇄 자료** — 한글학교·부모가 프린트해서 쓸 수 있는 자산. 저비용 고만족 채널
5. **번역 최소화** — 가능한 한국어를 한국어로 경험시키는 몰입 설계

## 4. Drops 분석

### 4.1 제품 개요
- **포지셔닝**: 5분 시각 기반 어휘 학습 앱 (성인 중심이지만 UX는 보편적)
- **가격**: 무료 (10시간마다 5분만) + $9.99~11/월, $69.99/년, $150 평생
- **소유**: Kahoot! 자회사 (2020년 인수)
- **핵심 철학**: *"짧고 강렬하게. 시간이 짧을수록 집중한다."*

### 4.2 온보딩 와이어프레임
```
[Screen 1] Hero 화면 — 시각적 인상
[Screen 2] 학습 언어 선택 (50+개, 스크롤 길음)
[Screen 3] 학습 이유 (여행/일/문화 등)
[Screen 4] 현재 실력 (초급/중급/고급)
[Screen 5] 일일 학습 시간 목표
[Screen 6] 푸시 알림 권한 요청
[Screen 7] 가입 (Google/Apple/이메일)
[Screen 8] 5분 프로모션 화면 ("5 free minutes every day!")
[Screen 9] 연령대 (UI 개인화 용도라는데 실제 영향 없음)
[Screen 10] 첫 레슨 시작
```

**비판 지점** (외부 UX 리뷰): 가치 제공 전 요구사항이 너무 많음. 10개 화면은 과함. *"Value first, requests second"* 원칙 위반.

### 4.3 홈 화면
- **상단**: 언어 전환, 스트릭, 시간 게이지
- **중앙**: 테마 선택 — 일러스트 타일 그리드
- **타일**: 테마명 + 단어 수 + 완료도
- **테마는 잠금 구조 (무료는 순차 해제, 프리미엄은 전체 오픈)**

### 4.4 레슨 화면 (Drops의 시그니처)
```
[5분 타이머 상단 상시 표시]

반복 사이클:
[패턴 A] 이미지 + 단어 페어 소개
 └─ 스와이프 업 → 다음

[패턴 B] 단어 → 이미지 매칭 (드래그)
 └─ 제한 시간 내 매칭

[패턴 C] 글자 조립 (철자)
 └─ 글자 타일을 올바른 순서로

[패턴 D] 단어 → 발음 듣기
 └─ 이미 학습한 단어 복습

타이머 종료 → 세션 종료
```

**설계 포인트**: 문제 유형이 빠르게 순환해서 지루할 틈 없음. 모든 인터랙션이 스와이프·드래그·탭 — 터치 친화적.

### 4.5 시각 언어
- **색상**: 단색 배경 + 미묘한 그라데이션 + 흰 텍스트 + 사진이 아닌 **벡터 일러스트**
- **일러스트**: 인하우스 디자이너가 모든 단어에 1:1 매칭되는 일러스트를 그림 (3,000+개)
- **타이포**: 대비 극대화, 한눈에 읽히는 구성
- **애니메이션**: 미묘한 모션 — 느리지 않고 빠르지도 않은 경쾌한 템포

### 4.6 지속 장치
- **시간 제약이 리텐션 메커닉**: 오늘 5분 못 채우면 내일 되돌아옴 (역설적 설계)
- **스트릭**: 연속 학습일
- **Scripts 앱** (별도): 비라틴 문자 학습 보조 — 한국어 한글에 해당

### 4.7 강점 / 약점

| 강점 | 약점 |
|---|---|
| 시각 디자인 업계 최고 수준 | 문법·문장 학습 없음 |
| 5분 세션의 집중도·중독성 | 어휘만 (대화 불가) |
| 스와이프·드래그 터치 UX | 온보딩 질문 너무 많음 |
| 미니멀 타이포·일러스트 | 성인 취향, 어린이에게 다소 밋밋 |
| Kahoot! 유통력 | 무료 플랜 제약 답답함 |

### 4.8 Drops에서 훔칠 것
1. **시각 언어 밀도** — 어린이 앱은 "화려하고 시끄럽게"가 정답이 아닐 수 있음. Drops의 미니멀이 오히려 Duolingo보다 집중 유발
2. **5분 타이머 HUD** — 시간 제약을 "여유"가 아니라 "미션"으로 재프레이밍
3. **스와이프·드래그 위주** — 어린이 손가락 조작에 친화적
4. **단어당 1:1 일러스트** — 사진보다 일러스트가 한국어 학습에 더 적합할 수 있음 (문화적 일반화)
5. **문제 유형 빠른 순환** — 지루함 방지

## 5. 보너스: Heritage 시장이 실제 소비하는 것

P4(재외동포) 가정이 현재 쓰는 것들. 제품 친숙도·정체성 측면에서 참고 필수.

### 5.1 한글이야호 (EBS, 2006~2011)
- **형태**: TV 애니메이션 (유튜브 전편 공개, 앱 아님)
- **강점**: 훈민정음 창제 원리 기반 한글 교수법. 음운 결합 원리 시각화. 북미 한글학교 학부모의 압도적 추천작.
- **약점**: 인터랙티브 아님, 2010년대 제작 그래픽, 진도 관리 불가
- **훔칠 것**: 자모 학습 커리큘럼 구조, 훈민정음 원리 기반 접근

### 5.2 뽀로로 한글놀이 (MIMIWORLD)
- **형태**: iOS/Android 앱 + 교구 연동
- **강점**: 뽀로로 IP 인지도, 유아 몰입도, 쓰기·스티커·퍼즐 인터랙션
- **약점**: 한글 자모 수준에서 끝남 (회화·문장 확장 없음), IP 의존, 한국어 UI
- **훔칠 것**: 유아에게 인기 있는 인터랙션 유형 (쓰기, 스티커, 퍼즐, 숨은 그림 찾기)

## 7. 우리의 고유 가치 제안 (UVP)

### 7.1 한 줄 UVP
> *"English-speaking kids' first Korean adventure — from Hangul to hello, in 5 minutes a day."*

### 7.2 차별화 6축

1. **한글부터 회화까지 연속**
   - 한글이야호는 한글만, Duolingo는 회화만. 우리는 Level 0~4로 연속.
   - 한 앱 안에서 5세~11세까지 자라남.

2. **한국 문화 맥락 통합**
   - 급식실, 생일, 명절, 지하철, 할머니 댁. 단어장 앱이 아닌 *경험 시뮬레이터*.
   - 진입점 분기: "우리 가족이 한국 사람이에요 / K-pop 좋아해요 / 그냥 배우고 싶어요" → 동일한 커리큘럼, 다른 스토리 래핑.

3. **어린이 전용 UX (5~11세)**
   - Duolingo의 딱딱함, Drops의 성인 미니멀함, 둘 다 배제.
   - DinoLingo 수준의 어린이 친화성 + Drops 수준의 시각적 밀도.

4. **치카치카 습관 엔진 이식**
   - 5~10분 세션, 매일 루틴, 성취 중심 보상(죄책감 X).
   - 캐릭터 성장·꾸미기(롤플레잉 요소)로 재방문 유도.

5. **상황극 기반 회화 (Level 3) — 우리만의 킬러 기능**
   - 음성 인식으로 역할 놀이. 어린이 발음 허용 범위 넓게.
   - "할머니께 전화하기", "급식 줄 서기", "K-pop 콘서트에서 팬레터 주기" 등 실제 장면.
   - Duolingo·DinoLingo·Drops 모두 없음.

6. **글로벌 접근 + 한국 특화의 공존**
   - 영어 UI 기본, 온보딩 모국어 선택 (중·일·베·러·스페인어 점진 추가).
   - 한국어 UI 모드(Phase 2)로 국내 다문화·중도입국 아동 확장 가능 — 같은 엔진.

### 7.3 Anti-Vision (우리가 **하지 않을 것**)
- 성인 대상 문법 강의
- K-pop 아이돌 IP 직접 라이선스 (저작권·비용·브랜드 리스크)
- 라이브 튜터링 (운영 부담)
- 소셜 랭킹·경쟁 (어린이 정신 건강)
- 한국어 UI only (글로벌 타겟 포기하는 셈)
- 뽀로로 수준의 자모 학습만 (천장이 낮음)

## 8. 구체적 목표 설정 (SMART)

### 8.1 프로덕트 마일스톤

| 시점 | 목표 |
|---|---|
| **M0~M1** | 온보딩·홈·레슨 코어 플로우 와이어프레임 확정 |
| **M2~M3** | Level 0 (한글 떼기 8개 레슨) + 캐릭터 컴패니언 MVP |
| **M4~M5** | Level 1 맛보기 (30단어, 3개 테마) + 부모 대시보드 최소 구현 |
| **M6** | 첫 상황극 시나리오 1개 (Level 3 맛보기) — 음성 인식 포함 |
| **M7~M9** | 베타 100명 (P4·P5 각 50명) 모집, 주간 피드백 루프 |
| **M10~M12** | App Store·Play Store 정식 출시 |

### 8.2 사용자·학습 KPI (MVP 종료 시점 기준)

| 지표 | 목표치 | 측정 근거 |
|---|---|---|
| 베타 유저 수 | 100명 (P4 50, P5 50) | 페르소나 분포 검증 |
| D1 리텐션 | 60% | 어린이 앱 평균 상회 |
| D7 리텐션 | 30% | 강한 초기 참여 |
| D30 리텐션 | 15% | 초기 PMF 시그널 |
| Level 0 완주율 | 30%+ | 한글 떼기 8레슨 완료자 비율 |
| 평균 세션 시간 | 5~10분 | 설계 의도 부합 |
| 주 3회 이상 활성 유저 | 전체 30%+ | "꾸준히" 지표 |
| 부모 대시보드 주 1회 접근 | 부모의 40% | Secondary User engagement |

### 8.3 학습 성과 지표 (정성)

- 베타 20가정 심층 인터뷰: "아이가 스스로 앱을 여는가?" — 50% 이상 Yes
- 한글 자모 인식 테스트: Level 0 완주자의 80%가 24자 중 20자 이상 식별
- 학부모 순추천지수(NPS): 40 이상

### 8.4 비즈니스 마일스톤 (Post-MVP)

| 시점 | 목표 |
|---|---|
| M12~M15 | 유료 구독 모델 테스트 (무료 vs $4.99/월 vs $39.99/년) |
| M15~M18 | 월 활성 사용자(MAU) 5,000명 |
| M18~M24 | 한국어 UI 모드 출시 → 국내 다문화 시장 파일럿 (교육청 1곳 B2G) |
| M24~M36 | MAU 50,000명, 연 매출 100만 달러 규모 |

### 8.5 3년 비전 선언
> *"재외동포 2·3세와 한국어 관심 글로벌 어린이가 한국어를 배우려 할 때 가장 먼저 떠올리는 앱. 그리고 국내 다문화 자녀가 한국 학교 생활에 적응하는 데 가장 의지하는 도구."*

## 9. 다음 단계

### 9.1 즉시 착수 가능

1. **와이어프레임 구체화 (다음 세션)**
   - 온보딩 10화면 플로우 (페르소나 분기 포함)
   - 홈 화면 (Level 0 선형 Path + Level 1+ 테마 그리드 혼합)
   - Level 0 한 레슨의 내부 플로우 (도입 → 학습 → 연습 → 평가 → 보상)
   - 부모 대시보드 (모바일 최소 화면)

2. **Level 0 커리큘럼 상세 설계**
   - 한글 자모 24개 + 기본 받침 → 8개 레슨으로 분해
   - 각 레슨 목표·학습 요소·미니게임 유형·평가 방식
   - 훈민정음 원리 기반 시각화 컨셉

3. **캐릭터·월드 세계관 컨셉**
   - 마스코트 후보: 호랑이·도깨비·토끼·곰 (한국 모티프)
   - 캐릭터 성장·꾸미기 메커니즘 스케치

### 9.2 선행 리서치 (병렬)

- 베타 타겟 인터뷰 계획 수립 (P4 5가정, P5 5가정)
- 경쟁 앱 직접 사용 + 스크린 레코딩 (본 분석 보강)
- 음성 인식 API 기술 조사 (어린이 한국어 발음 인식 품질)

### 9.3 의사결정 대기 중

- 1차 마스코트·캐릭터 방향 (한국 전래 모티프 vs 창작 캐릭터)
- 기술 스택 (React Native / Flutter / Kotlin+Swift)
- 음성 인식 (자체 / Google / OpenAI Whisper / Clova)
- 수익 모델 세부 (무료 범위, 유료 범위)
